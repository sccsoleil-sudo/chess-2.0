# Chess 2.0

A clean Vite + React + TypeScript chess game.

## Features

- Legal move validation with `chess.js`
- Select-and-move board interaction
- Check, checkmate, draw, and turn status
- Captured pieces and material advantage
- Move history, undo, reset, and board flip controls

## Scripts

```bash
npm install
npm run dev
npm run build
```
